# SaveMyMoney Web App

  

A web app to help users to track monthly transactions such as expenses and incomes. Created using HTML, CSS, Javascript Vanilla and Firebase.

  

## Demo

  

Here you can find the app demo: https://savemymoney.vercel.app/

  

## Built with

* HTML

* CSS

* JavaScript (Vanilla)

* Firebase
